\shici{五言绝句}{鹿柴}{唐}{王维}{ %
空山不见人，但闻人语响。\\
返景入深林，复照青苔上。
}

\shici{五言绝句}{竹里馆}{唐}{王维}{ %
独坐幽篁里，弹琴复长啸。\\
深林人不知，明月来相照。
}

\shici{五言绝句}{送别}{唐}{王维}{ %
山中相送罢，日暮掩柴扉。\\
春草明年绿，王孙归不归？
}

\shici{五言绝句}{相思}{唐}{王维}{ %
红豆生南国，春来发几枝？\\
愿君多采撷，此物最相思。
}

\shici{五言绝句}{杂诗}{唐}{王维}{ %
君自故乡来，应知故乡事。\\
来日绮窗前，寒梅著花未？
}

\shici{五言绝句}{送崔九}{唐}{裴迪}{ %
归山深浅去，须尽丘壑美。\\
莫学武陵人，暂游桃源里。
}

\shici{五言绝句}{终南望馀雪}{唐}{祖咏}{ %
终南阴岭秀，积雪浮云端。\\
林表明霁色，城中增暮寒。
}

\shici{五言绝句}{宿建德江}{唐}{孟浩然}{ %
移舟泊烟渚，日暮客愁新。\\
野旷天低树，江清月近人。
}



\shici{五言绝句}{春晓}{唐}{孟浩然}{ %
春眠不觉晓，处处闻啼鸟。\\
夜来风雨声，花落知多少？
}


\shici{五言绝句}{夜思}{唐}{李白}{ %
床前明月光，疑是地上霜。\\
举头望明月，低头思故乡。
}

\shici{五言绝句}{怨情}{唐}{李白}{ %
美人卷珠帘，深坐蹙蛾眉。\\
但见泪痕湿，不知心恨谁？
}

\shici{五言绝句}{八阵图}{唐}{杜甫}{ %
功盖三分国，名成八阵图。\\
江流石不转，遗恨失吞吴。
}

\shici{五言绝句}{登鹳雀楼}{唐}{王之涣}{ %
白日依山尽，黄河入海流。\\
欲穷千里目，更上一层楼。
}

\shici{五言绝句}{送灵澈}{唐}{刘长卿}{ %
苍苍竹林寺，杳杳钟声晚。\\
荷笠带斜阳，青山独归远。
}

\shici{五言绝句}{弹琴}{唐}{刘长卿}{ %
泠泠七弦上，静听松风寒。\\
古调虽自爱，今人多不弹。
}

\shici{五言绝句}{送上人}{唐}{刘长卿}{ %
孤云将野鹤，岂向人间住！\\
莫买沃洲山，时人已知处。
}

\shici{五言绝句}{秋夜寄邱员外}{唐}{韦应物}{ %
怀君属秋夜，散步咏凉天。\\
空山松子落，幽人应未眠。
}

\shici{五言绝句}{听筝}{唐}{李端}{ %
鸣筝金粟柱，素手玉房前。\\
欲得周郎顾，时时误拂弦。
}

\shici{五言绝句}{新嫁娘}{唐}{王建}{ %
三日入厨下，洗手作羹汤。\\
未谙姑食性，先遣小姑尝。
}

\shici{五言绝句}{玉台体}{唐}{权德舆}{ %
昨夜裙带解，今朝蟢子飞。\\
铅华不可弃，莫是藁砧归。
}


\shici{五言绝句}{江雪}{唐}{柳宗元}{ %
千山鸟飞绝，万径人踪灭。\\
孤舟蓑笠翁，独钓寒江雪。
}

\shici{五言绝句}{行宫}{唐}{元稹}{ %
寥落古行宫，宫花寂寞红。\\
白头宫女在，闲坐说玄宗。
}

\shici{五言绝句}{问刘十九}{唐}{白居易}{ %
绿蚁新醅酒，红泥小火炉。\\
晚来天欲雪，能饮一杯无？
}

\shici{五言绝句}{何满子}{唐}{张祜}{ %
故国三千里，深宫二十年。\\
一声何满子，双泪落君前。
}

\shici{五言绝句}{登乐游原}{唐}{李商隐}{ %
向晚意不适，驱车登古原。\\
夕阳无限好，只是近黄昏。
}

\shici{五言绝句}{寻隐者不遇}{唐}{贾岛}{ %
松下问童子，言师采药去。\\
只在此山中，云深不知处。
}

\shici{五言绝句}{渡汉江}{唐}{李频}{ %
岭外音书绝，经冬复立春。\\
近乡情更怯，不敢问来人。
}

\shici{五言绝句}{春怨}{唐}{金昌绪}{ %
打起黄莺儿，莫教枝上啼。\\
啼时惊妾梦，不得到辽西。
}

\shici{五言绝句}{哥舒歌}{唐}{西鄙人}{ %
北斗七星高，哥舒夜带刀。\\
至今窥牧马，不敢过临洮。
}

\shici{五言绝句}{长干行二首之一}{唐}{崔颢}{ %
君家何处住，妾住在横塘。\\
停船暂借问，或恐是同乡。
}

\shici{五言绝句}{长干行二首之二}{唐}{崔颢}{ %
家临九江水，来去九江侧。\\
同是长干人，生小不相识。
}

\shici{五言绝句}{玉阶怨}{唐}{李白}{ %
玉阶生白露，夜久侵罗袜。\\
却下水晶帘，玲珑望秋月。
}

\shici{五言绝句}{塞下曲四首之一}{唐}{卢纶}{ %
鹫翎金仆姑，燕尾绣蝥弧。\\
独立扬新令，千营共一呼。
}

\shici{五言绝句}{塞下曲四首之二}{唐}{卢纶}{ %
林暗草惊风，将军夜引弓。\\
平明寻白羽，没在石棱中。
}

\shici{五言绝句}{塞下曲四首之三}{唐}{卢纶}{ %
月黑雁飞高，单于夜遁逃。\\
欲将轻骑逐，大雪满弓刀。
}

\shici{五言绝句}{塞下曲四首之四}{唐}{卢纶}{ %
野幕蔽琼筵，羌戎贺劳旋。\\
醉和金甲舞，雷鼓动山川。
}

\shici{五言绝句}{江南曲}{唐}{李益}{ %
嫁得瞿塘贾，朝朝误妾期。\\
早知潮有信，嫁与弄潮儿。
}




\shici{七言绝句}{回乡偶书}{唐}{贺知章}{ %
少小离家老大回，乡音无改鬓毛衰。\\
儿童相见不相识，笑问客从何处来？
}

\shici{七言绝句}{桃花溪}{唐}{张旭}{ %
隐隐飞桥隔野烟，石矶西畔问渔船。\\
桃花尽日随流水，洞在清溪何处边？
}

\shici{七言绝句}{九月九日忆山东兄弟}{唐}{王维}{ %
独在异乡为异客，每逢佳节倍思亲。\\
遥知兄弟登高处，遍插茱萸少一人。
}

\shici{七言绝句}{芙蓉楼送辛渐}{唐}{王昌龄}{ %
寒雨连江夜入吴，平明送客楚山孤。\\
洛阳亲友如相问，一片冰心在玉壶。
}





\shici{唐诗}{过故人庄}{唐}{孟浩然}{ %
故人具鸡黍， 邀我至田家。 \\
绿树村边合， 青山郭外斜。 \\
开轩面场圃， 把酒话桑麻。 \\
待到重阳日， 还来就菊花。 \\
}


\shici{唐诗}{小儿垂钓}{唐}{胡令能}{ %
蓬头稚子学垂纶， \\
侧坐莓台草映身。 \\
路人借问遥招手， \\
怕得鱼惊不应人。 \\
}



\shici{唐诗}{山村咏怀}{宋}{邵康节}{ %
一去二三里， 烟村四五家。 \\
亭台六七座， 八九十枝花。
}
